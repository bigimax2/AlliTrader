# Система алертов по количеству предметов

## Что реализовано

### Новые модели (`trader/models.py`)
- `AlertThreshold` - пороги алертов для каждого пользователя и конкретного предмета
  - `user_id` - ID пользователя
  - `type_id` - ID типа предмета
  - `min_quantity` - минимальное количество для алерта

### Новые представления (`trader/views.py`)
- `alert_settings()` - страница настройки порогов для пользователя
- `delete_threshold()` - удаление порога через AJAX

### Новые формы (`trader/forms.py`)
- `AlertThresholdForm` - форма добавления/редактирования порога

### Новые шаблоны
- `trader/templates/alert_settings.html` - страница настройки порогов
- Обновлён `trader/templates/render_traders.html` - алерты в таблице

## Логика алертов

При отображении ассетов:
- Сравниваем `asset.quantity` с порогом `min_quantity`
- **Красный цвет (critical)**: количество < 25% от порога
- **Жёлтый цвет (warning)**: количество < 50% от порога, но >= 25%
- **Обычный цвет**: количество >= 50% от порога

## Как использовать

### Для пользователя:

1. Перейти на страницу **Настройки алертов** (ссылка в боковом меню или по пути `/trader/alert_settings/`)
2. Выбрать предмет из списка и установить минимальное количество
3. При отображении ассетов предметы с низким количеством будут подсвечены:
   - Жёлтым цветом - если количества меньше 50% от порога
   - Красным цветом - если количества меньше 25% от порога

### Для администратора:

- Доступ в админку Django: `/admin/trader/alertthreshold/`
- Можно просматривать и управлять всеми настройками алертов

## Структура URL

```
/trader/alert_settings/     - страница настройки алертов
/trader/delete_threshold/   - AJAX endpoint для удаления порога
```

## Примеры использования

### Пример 1: Предупреждение о малом количестве топлива
- Предмет: "Hydrogen Fuel Cells"
- Установлен порог: 10000
- Если есть 3000 - красный (30% от порога)
- Если есть 4000 - жёлтый (40% от порога)
- Если есть 6000 - нормально (60% от порога)

### Пример 2: Критическая ситуация с аmmo
- Предмет: "Heavy Missile Ammo"
- Установлен порог: 500
- Если есть 100 - красный (20% от порога)
- Если есть 250 - жёлтый (50% от порога)
