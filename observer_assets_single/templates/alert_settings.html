Как это работает

Эта страница позволяет настроить индивидуальные пороги количества предметов для получения алертов в таблице ассетов.

  • Красный цвет (critical): количество предметов меньше или равно 25% от порога
  • Желтый цвет (warning): количество предметов меньше или равно 50% от порога
  • Нормальный цвет: количество больше 50% от порога

Например, если вы установили порог 100 для предмета:

  • Количество ≤ 25 - красный цвет (critical)
  • Количество ≤ 50 - желтый цвет (warning)
  • Количество 51-100 - желтый цвет (warning, близко к порогу)
  • Количество 101+ - нормальный цвет

Важно: порог алерта должен быть больше 0.

Импорт алертов из файла
Загрузите JSON-файл с резервной копией алертов. Дубликаты будут обновлены, новые - добавлены. Пороги ≤ 0 будут пропущены.
Важно: файл должен содержать корректный JSON с полем "alerts" (список объектов с type_id и min_quantity > 0)
Типичные ошибки импорта:
  • Файл не имеет расширения .json
  • Содержимое не является валидным JSON
  • Отсутствует поле "alerts" или оно не является списком
  • type_id или min_quantity не являются числами
